# About NingClaw Software Workflow Lab

NingClaw Software Workflow Lab is a Chinese-language content property focused on overseas SaaS and workflow tools.

It is built for:
- solo operators
- small teams
- marketers and content operators
- founders comparing software before buying

## What this site is
A practical review and workflow lab. We publish:
- product evaluation notes
- implementation examples
- buyer checklists
- workflow-focused tutorials

## What this site is not
- a coupon spam directory
- a fake-media property
- a claims-heavy affiliate funnel
- a company with invented team size or invented usage history

## How we work
We use official vendor materials, public documentation, and careful comparison. When an affiliate link is used, we say so clearly.
